# personal-site

Source code for my personal website/portfolio.

## Structure

* `index.html` – homepage
* `about.html` – about page
* `music.html` – music page
* `css/` – stylesheets
* `js/` – scripts
* `assets/` – album covers, images and other static assets

## Deployment

This site is deployed using Cloudflare Pages.
